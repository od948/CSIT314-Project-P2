Compilation instructions.

ensure java version 11.0.7 or greater is installed.
//on command line write java -version to see current version.

download and install intelliJ ide community edition: https://www.jetbrains.com/idea/download/#section=windows

after installing, open intelliJ and click file->open then select the folder containing the project and open.

to compile click the hammer icon on the toolbar at the top, then click the play button to run the program.

the output will be in the command prompt at the bottom of the window.

optionally there is a precompiled jar in the folder "out", using command line, enter the directory where the jar is and run the jar using "java -jar csit314p2.jar"
